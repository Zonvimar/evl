# Система управления библиотекой

Простое веб-приложение для экзамена по веб-программированию.

## Функционал

- Авторизация пользователя
- Просмотр всех книг с пагинацией
- Добавление новых книг
- Удаление книг по ID
- Сортировка по всем полям
- Адаптивный дизайн

## Структура файлов

```
├── index.php      # Главная страница с авторизацией
├── config.php     # Подключение к БД
├── view.php       # Просмотр книг
├── add.php        # Добавление книг
├── delete.php     # Удаление книг
├── logout.php     # Выход
└── database.sql   # SQL для создания БД
```

## Установка

1. Загрузи все файлы на сервер
2. Создай БД `library_exam` в phpMyAdmin
3. Выполни SQL из `database.sql`
4. В `config.php` замени данные подключения к БД:
   ```php
   $username = 'твой_логин_от_бд';
   $password = 'твой_пароль_от_бд';
   ```

## База данных

Таблица `books`:
- id (AUTO_INCREMENT)
- title (название)
- author (автор)
- price (цена)
- year_published (год издания)
- genre (жанр)
- page_count (страницы)
- publisher (издательство)

В базе уже есть 18 тестовых записей.
